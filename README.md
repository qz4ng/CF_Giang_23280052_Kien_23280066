# TÍNH TOÁN TÀI CHÍNH
Đồ án giữa kỳ môn tính toán tài chính 
Mục đích: Sử dụng máy học và kiến thức tài chính để xây dựng mô hình dự đoán giá chứng khoán (ACB.VN) từ đó đưa ra quyết định: bán, mua hay giữ
## Một số kiến thức được sử dụng

## Cách cài đặt và hoạt động 
- Class StockData: tải dữ liệu chứng khoán của ACB.VN, lấy thông tin cơ bản (Tên, ngành, chỉ số PE, PB, EPS, ROE,..) và báo cáo tài chính (income statement, balance sheet, cash flow)
- Class Processing: cleaning, fundamental analysis
Đổi với class Fundamental: dùng để tính các chỉ số tài chính
Yêu cầu dữ liệu nhập vào có dạng:
|NetIncome|
- Class Analysis: regression, time series, volatility
- Class Trading stategies
- Class Visualization
